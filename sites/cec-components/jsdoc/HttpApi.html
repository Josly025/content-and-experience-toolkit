HttpApi

new HttpApi()

Define the HTTPS NodeJS implementation requests for calls to the Language Service Provider.

Methods

callService(type, targetURL, restArgs) → {Promise.<object>}

Make the NodeJS HTTPS call to the Language Service Provider.
Parameters:
Name Type Description
type "GET" | "POST" The type of HTTP request to make.
targetURL string The Language Service Provider URL to call.
restArgs object The arguments to provide in the HTTPS request.
Properties
Name Type Attributes Description
data object Data to include in a POST request.
beforeSend function <optional>
 A function to call before sending the request. This allows the caller to update any additional information after the request has been created.
authToken string <optional>
 The "Authorization" header parameter to include in the request.
timeout string <optional>
 A timeout for the request.
Returns:
A Promise that resolves with the Language Service Provider response to the request.
